选题

Topic Selection

框架价值

  • 维持内容系列感
  • 增强创作持续性
  • 平衡各板块占比
  • 便于归档管理

执行流程

  • 开放选题池收集灵感
  • @Yu 每周整理归档
  • 周选题会脑暴+优先级
  • 任务认领 → Promise
  • 会后公开执行计划
产出
灵感 + 内容形式 + 实现方案 + 本周优先级 + 负责人认领
02
⌂

制作

Production

核心机制

  • 飞书机器人群内公开管理
  • 社媒内容面板沉淀文案
  • 渠道负责人自主更新状态
  • 机器人同步状态变更

状态管理

  • 实时状态可见
  • 阻塞点及时暴露
  • 进度透明可追溯
关键原则
自驱更新 · 公开透明 · 飞书联动
03
◎

审核

Review

审核要点

  • 内容质量：准确性 + 清晰度
  • 品牌一致性：调性 + 视觉
  • 合规检查：敏感词 + 版权
  • Context 对齐

提交流程

  • 更新状态为「待审核」
  • 内容卡片转发社媒群
  • 含：完整文案 + 视觉素材
✓ 通过 → 定时间/封面/平台
↻ 修改 → 标注意见返回
✕ 驳回 → 重新评估选题
04
◈

发布

Distribution

发布节奏

  • 首发：主平台按时发布
  • 分发：首发后 48h 内完成
  • 渠道负责人发布后群内同步

闭环机制

  • 数据追踪与整理
  • 协作效果复盘
  • 周报同步全员
  • 反哺下周选题
周报内容
内容数据 · 协作亮点 · Delay 记录 · 改进建议

工具链

飞书机器人 社媒内容面板 选题池 内容框架表

关键角色

@Yu 选题整理 渠道负责人 审核决策人

核心节奏
